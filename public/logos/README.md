# Logos de la Aplicación

Esta carpeta contiene los logos estáticos de la aplicación.

## Archivos requeridos:

- `kainet-logo.png` - Logo de Kainet (empresa)
- `kaido-logo.png` - Logo de Kaido (aplicación)

## Especificaciones recomendadas:

### Kainet Logo
- Formato: PNG con transparencia
- Tamaño recomendado: 200x200px (mínimo)
- Resolución: 2x para pantallas retina (400x400px)
- Fondo: Transparente

### Kaido Logo
- Formato: PNG con transparencia
- Tamaño recomendado: 200x200px (mínimo)
- Resolución: 2x para pantallas retina (400x400px)
- Fondo: Transparente
- Variantes: 
  - Horizontal (para headers)
  - Vertical/Isotipo (para favicon)

## Uso en la aplicación:

Los logos se referencian desde el código como:
- `/logos/kainet-logo.png`
- `/logos/kaido-logo.png`
